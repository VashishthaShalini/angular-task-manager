# 📝 Angular Task Manager

A clean and interactive Task Manager built using Angular (Standalone Components).

## 🚀 Features

- ✅ Add & Delete Tasks  
- ✔ Mark Tasks as Completed  
- 🔍 Search Tasks (Real-time)  
- 🎯 Filter Tasks (All / Completed / Pending)  
- ✏ Edit Tasks Inline  
- 💾 Local Storage Support (Data persists after refresh)  

## 🛠 Tech Stack

- Angular (Standalone Components)
- TypeScript
- HTML / CSS
- LocalStorage API

## ⚡ How to Run

```bash
npm install
ng serve
